# 🛍️ Retail Database Management & Analysis (SQL)

## 📌 Overview

This project simulates a real-world retail database and explores how SQL can be used to extract meaningful business insights.

From designing the database structure to running advanced queries, this project focuses on understanding **customer behavior, revenue trends, and inventory performance**.

---

## 🧠 What I Did

* Built a relational database from scratch (customers, orders, inventory, shipments)
* Wrote advanced SQL queries using joins, subqueries, and aggregations
* Performed database modifications (ALTER, UPDATE, DELETE)
* Created business-focused metrics to support decision-making

---

## 📊 Key Insights & Analyses

* 💰 **Customer Lifetime Value** – Identified high-value customers based on total spend
* 📍 **Revenue by State & Category** – Analyzed where revenue is coming from
* 📦 **Inventory Monitoring** – Flagged low-stock and out-of-stock items
* 🧩 **Customer Segmentation** – Grouped customers using spending behavior (CASE statements)
* 📈 **Order Value Metrics** – Calculated average and total order performance
